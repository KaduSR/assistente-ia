# Assistente IA 🤖

> Chatbot com inteligencia artificial, streaming de respostas e contexto persistente.

## Stack
Next.js • OpenAI API • Python (embeddings) • SSE • Vector Store

## Funcionalidades
- Streaming de respostas (Server-Sent Events)
- Contexto de conversa com vector store
- Upload e analise de documentos (PDF/TXT)
- Historico de conversas persistente
- Respostas em tempo real

## Metrica
Primeira resposta em menos de 500ms com streaming continuo.
